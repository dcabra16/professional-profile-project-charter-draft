📋 1. Project Title & Description

Professional Profile Website

My project aims to create a personal profile website that addresses the challenge of effectively showcasing my professional skills and experiences online. By delivering a well-designed and informative website, I aim to showcase my personal brand in a professional way, giving potential employers, peers, and colleagues an accessible view of my skills and accomplishments. This project will benefit anyone looking to learn more about my professional background and connect with me for potential opportunities.

📏 4. High-Level Scope

✅ In Scope for Semester Project

  • A homepage with a personal introduction to viewers.
  • An about me section.
  • A portfolio/projects section that showcases projects from my education with short descriptions.
  • A downloadable PDF version of my resume
  • A contact form or link that will redirect users to send me an email/my LinkedIn page
  • A responsive visual layout (fonts will match, and the page will be eligible for users to understand).

🗺️ Roadmap Items (≤6 Months)

  • Optimize meta tags, alt text, and content to improve search engine visibility.
  • Set up goals to integrate detailed analytics tools.
  • Revamp the design for a modern look.
  • Enhance navigation features for a better user experience.
  • Add social media sharing buttons to remain relevant to visitors over time.

❌ Out of Scope

  • It should not function like an e-commerce website.
  • It should not function like a blog.
  • It should not have user registration or features for social interaction with others.
  • It should not be created for marketing materials or promoting content.
  • It should not have live streaming or video conference capabilities.
